# Personal Expense Tracker

A lightweight desktop application built with Python and PyQt6 designed to log, filter, and track monthly and yearly expenditures.

## Features

- **Multi-Year Tracking:** Filter and view expense data across multiple months and years.
- **Dynamic Summaries:** Displays real-time calculations for both selected monthly and yearly totals.
- **Centered UI Design:** Input forms and table rows are uniformly centered for optimal visual scannability.
- **Local Data Persistence:** Saves records automatically to a hidden JSON file in your home directory.

## Prerequisites

- Python 3.10 or newer
- PyQt6
